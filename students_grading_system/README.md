## Student Grading Management System
This is a command-line Student Grading Management System built with Python.
The project was built as a learning exercise after completing a Python Expense Tracker project. The goal was to practise designing a larger application involving students, teachers, grading, CRUD operations, and JSON persistence.

## Motivation
When I checked for projects to build online, the ones I saw won't give me as much joy as I wanted and at the same time let me practise what I just learnt. 
Then I remembered my reports cards from school when I was in Primary school and was motivated by it to recreate it.

## Features
+ Student creation and management
+ Student transfer between classes
+ Teacher creation and assignment
+ Score entry and editing
+ Report card generation
+ JSON data persistence
+ Student lookup using unique IDs
+ Teacher lookup using unique IDs

## Technologies Used
+ Python
+ JSON
+ Tabulate
+ OS
+ Secrets

## How To Run
1. Clone repository
2. Install dependencies
3. Run index.py

## Data Structure Design
+ Classes contain students
+ Students contain subjects
+ Subjects contain scores

## Lessons Learned
- Working with nested dictionaries
- JSON file persistence
- CRUD operations
- Input validation
- Debugging complex systems
- Refactoring duplicated logic
